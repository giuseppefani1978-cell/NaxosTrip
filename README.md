# 🏝️ Naxos Interactive Travel Map

Une carte interactive de l'île de **Naxos** réalisée avec **Leaflet** et **OpenStreetMap**, optimisée pour iPhone.

Le projet permet de préparer un séjour depuis **Palatiana Agriturismo (Galini)** et de comparer rapidement les temps de trajet en voiture ou en scooter.
